# Example - No Candidate From Clean Validation

## Episode

A small project story updated one scout index entry, regenerated methodology
surfaces, and passed all required checks. `/validate` found no material
findings, no user correction, no repeated friction, and no missing guardrail.

## Learning Review Output

```text
RESULT: no-candidate
Reason: The episode completed through the existing story loop without a reusable
workflow correction or repeated process friction.
Evidence checked: story acceptance criteria, git diff, validation report,
methodology-check, lint
```

## Why No Draft Is Created

The clean run is useful evidence that the existing workflow worked, but it does
not identify a durable correction. Drafting a candidate here would create
reflection overhead without reducing future work.
